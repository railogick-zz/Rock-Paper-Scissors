Description

People love to see their results as they're running to their goal.
So, let's write a rating system:

When the game starts, the program should read ratings from a file named rating.txt and ask the user for the name.
You can see the file format below. It's just name and a score divided by a single space.

Tim 350
Jane 200
Alex 400

The program should print user rating when the user enters !rating. For example, if your rating is 0 then the program should print:

Your rating: 0

Make it so for every game you add 50 points to the player for every draw and 100 for every win, and nothing for losing.

At the start of the game, the player should enter his/her name. Make sure that the name does not contain spaces! After that, the program should welcome the player and start playing with him.

Examples

Example 1

Enter your name: Tim
Hello, Tim
!rating
Your rating: 350
rock
Sorry, but computer chose paper
rock
Well done. Computer chose scissors and failed
!rating
Your rating: 450
!exit
Bye!
