Description

Well, let's do something real. Nobody wants the game where you always lose.
Using the power of the random module, we'll do a really interesting game.

You should write a program that reads input from the user, chooses a random option and then says the result.
There are a few examples below, providing output for any outcome (where options are rock, paper or scissors):

  1. Lose -> Sorry, but computer chose <option>
  2. Draw -> There is a draw (<option>)
  3. Win -> Well done. Computer chose <option> and failed
