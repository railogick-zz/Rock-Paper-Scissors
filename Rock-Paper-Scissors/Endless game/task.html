Description

Actually, that was cool!

But it was really short nobody plays just a single game of rock-paper-scissors. We need to do some literally unstoppable game for unstoppable players. Not really unstoppable, implement a way to stop your program.

Improve your program so it wouldn't stop until the user enters !exit. After entering !exit the program should print Bye! and terminate. Optionally, you can add a !help command that helps the user to understand the rules, but it won't be tested.

Example

rock
Well done. Computer chose scissors and failed
paper
Well done. Computer chose rock and failed
paper
There is a draw (paper)
scissors
Sorry, but computer chose rock
!exit
Bye!
