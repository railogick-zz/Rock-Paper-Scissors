Description

Rock, paper, scissors is a well-known hand game. Each one of two players forms one of three (or more) shapes with hand.
The game is widely used to make a fair decision between equal options.

So, let's start with an unfair version :)

Write a program that reads input from the user and chooses a winning option.
As an example, if the user enters rock, the program should print Sorry, but computer chose paper and so on.

Examples

Example 1

scissors
Sorry, but computer chose rock

Example 2

paper
Sorry, but computer chose scissors
